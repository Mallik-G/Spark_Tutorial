## Scala Data Types:
* In Scala, every data type is an Object
* Scala has the following data types:
  * Byte (1 Byte)
  * Short (2 Bytes)
  * Int (4 Bytes)
  * Long (8 Bytes)
  * Float (4 Bytes)
  * Double (8 Bytes)
  * Char
  * String
  * Boolean
  * Unit
  * Null
  * Any
  * AnyRef
* Scala Literals:
  * Integer:
    10, 200L
  * Floating:
    1.0f, 0.0
  * Boolean:
    true/false
  * Char:
    'a', 'b'
  * String:
    * Single line:
      "Ravi"
    * Multi line:
      """This scala
      Tutorial.
      """
